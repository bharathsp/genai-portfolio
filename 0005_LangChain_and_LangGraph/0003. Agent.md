# 🦾 Agents

An **AI Agent** is an AI system that can **understand a goal, use tools, and perform tasks**.

A simple way to think about it is:

> **🧠 AI Agent = LLM (Model) + 🛠️ Tools**

---

## 🤖 What is a Basic Agent?

A **basic agent** is simply:

### 🧠 LLM + 🛠️ Tool

The **LLM (Large Language Model)** provides the intelligence:

* Understands the user's request
* Reasons about what to do
* Decides which tool to use

The **tool** allows the agent to interact with the outside world:

* 🌐 Search the web
* 📧 Send emails
* 📅 Create calendar events
* 💻 Execute code
* 💾 Query databases

### Example

👤 User:

> "What is today's weather in Chennai?"

🧠 LLM:

> "I need current weather information."

🛠️ Tool:

> Calls a weather API.

✅ Agent:

> Returns the latest weather report.

---

# 🔧 What is `model.bind_tools`?

In many AI frameworks (such as LangChain), you may see:

```python
model.bind_tools(...)
```

### Simple meaning:

It **connects tools to the AI model**.

Without binding tools:

🧠 LLM:

> "I can only answer from my knowledge."

With binding tools:

🧠 LLM:

> "I can use a calculator, search the web, or call APIs when needed."

### Real-world analogy

👨‍🔧 Worker = LLM

🧰 Toolbox = Tools

`model.bind_tools()` = Giving the worker access to the toolbox.

### Example
## 🤔 Without `bind_tools`

The AI can only **talk**.

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(model="gpt-4o-mini")

response = model.invoke("What is 25 * 40?")
print(response.content)
```

🧠 The LLM answers from its own reasoning.

---

# 🛠️ With `model.bind_tools(...)`

Suppose we create a simple calculator tool.

```python
from langchain.tools import tool

@tool
def multiply(a: int, b: int) -> int:
    """Multiply two numbers."""
    return a * b
```

Now, we give this tool to the model.

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(model="gpt-4o-mini")

# Bind the tool to the model
agent = model.bind_tools([multiply])
```

Now the AI has access to the calculator.

```python
response = agent.invoke(
    "What is 25 multiplied by 40?"
)

print(response)
```

### What happens internally?

```text
👤 User
   │
   ▼
🧠 LLM receives the question
   │
   ▼
🤔 "I have a multiply tool."
   │
   ▼
🛠️ Calls:
   multiply(25, 40)
   │
   ▼
✅ Tool returns:
   1000
   │
   ▼
🤖 AI responds:
   "25 multiplied by 40 is 1000."
```

# 🎯 Easy Formula to Remember

```text
📚 Data
    ↓
🧠 LLM (Model)
    +
🛠️ Tools
    ↓
🦾 Basic AI Agent
    ↓
⚙️ Automation Platforms (like n8n)
    ↓
🚀 Real-world AI Applications
```

**One-line summary:**

> **A basic AI agent is an LLM that has access to tools, and `model.bind_tools()` is the mechanism that gives those tools to the model. Platforms like n8n orchestrate agent workflows, while OpenAI Codex specializes in coding-related agent tasks.**
