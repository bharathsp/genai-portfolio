# 📝 Text Prompt vs 💬 Message Prompt in LangChain

When interacting with LLMs in LangChain, there are two common ways to send instructions:

```text
1️⃣ Text Prompt
2️⃣ Message Prompt
```

Understanding this difference is very important because modern chat models (GPT, Gemini, Claude, Llama, etc.) internally work with **messages**, not just plain text.

---

# 🎯 What is a Prompt?

A prompt is simply:

> 🗣️ The input you give to an LLM.

Example:

```text
Explain Agentic AI.
```

This is a prompt.

---

# 1️⃣ 📝 Text Prompt

## What is it?

A **Text Prompt** is a single string sent to the model.

---

## Example

```python
prompt = """
Explain Agentic AI in simple terms.
"""
```

---

## Using LangChain

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template(
    "Explain {topic} in simple terms"
)

formatted_prompt = prompt.invoke(
    {"topic":"LangGraph"}
)

print(formatted_prompt)
```

Output:

```text
Explain LangGraph in simple terms
```

---

## Visual Representation

```text
┌──────────────────────┐
│ Explain LangGraph    │
└──────────────────────┘
```

One block of text.

---

## Real-Life Example

Teacher asks:

```text
Explain photosynthesis.
```

Single instruction.

No conversation history.

---

# 💬 2️⃣ Message Prompt

## What is it?

A **Message Prompt** is a collection of messages.

Instead of:

```text
One Text Block
```

You create:

```text
System Message
Human Message
AI Message
```

just like a real conversation.

---

## Example

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate.from_messages(
    [
        (
            "system",
            "You are an AI expert"
        ),

        (
            "human",
            "Explain LangGraph"
        )
    ]
)
```

---

## Visual Representation

```text
📋 System
You are an AI expert

        ↓

👤 Human
Explain LangGraph
```

---

## Output

```python
messages = prompt.invoke({})
```

Produces:

```python
[
 SystemMessage(...),
 HumanMessage(...)
]
```

---

# Why Message Prompts?

Modern LLMs are chat-based.

They understand:

```text
📋 System Instructions

👤 User Questions

🤖 Previous AI Responses

🛠️ Tool Results
```

much better than one giant text string.

---

# Text Prompt vs Message Prompt

## Text Prompt

```text
You are an AI expert.

Explain LangGraph.
```

Everything is merged into one text block.

---

## Message Prompt

```text
📋 System
You are an AI expert.

👤 Human
Explain LangGraph.
```

Roles are preserved.

# 🧠 ChatPromptTemplate

Most important class for Message Prompts.

---

## Example

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate.from_messages(
    [
        (
            "system",
            "You are a helpful assistant."
        ),

        (
            "human",
            "{question}"
        )
    ]
)
```

---

### Invoke

```python
messages = prompt.invoke(
    {
        "question":"What is LangGraph?"
    }
)
```

---

### Result

```text
📋 System
You are a helpful assistant.

👤 Human
What is LangGraph?
```

# 🤖 Why Agents Use Message Prompts

Agentic AI systems need:

```text
📋 System Instructions

👤 User Messages

🤖 AI Thoughts

🛠️ Tool Results

💾 Conversation History
```

This is impossible to manage elegantly using one giant text prompt.

Message prompts solve this problem.

---

# 🚀 Typical LangChain Workflow

```python
prompt = ChatPromptTemplate.from_messages(...)

chain = (
    prompt
    | model
)
```

---

Flow:

```text
💬 ChatPromptTemplate
          │
          ▼
     Messages
          │
          ▼
      🤖 LLM
          │
          ▼
      Response
```

# 📊 Comparison Table

| Feature              | 📝 Text Prompt     | 💬 Message Prompt |
| -------------------- | ------------------ | ----------------- |
| Format               | Single string      | Multiple messages |
| Roles                | ❌ No               | ✅ Yes             |
| System Instructions  | ⚠️ Mixed with text | ✅ Separate        |
| Conversation History | ❌ Difficult        | ✅ Easy            |
| Tool Calling         | ❌ Limited          | ✅ Excellent       |
| Agents               | ❌ Rarely used      | ✅ Standard        |
| Modern LLMs          | ⚠️ Works           | ✅ Preferred       |

# Interview Answer 🎤

> **A Text Prompt (`PromptTemplate`) sends a single text string to the model, while a Message Prompt (`ChatPromptTemplate`) sends structured messages with roles such as System, Human, AI, and Tool. Message prompts are preferred for modern chat models, agents, tool calling, and multi-turn conversations because they preserve conversational structure and context.** 🚀💬
