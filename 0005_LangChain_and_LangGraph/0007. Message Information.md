# 💬 Message Information in LangChain

Every message in LangChain (System, Human, AI, Tool) contains important information.

The three most common attributes are:

```text
📋 role
📝 content
🏷️ metadata
```

Think of a message like a WhatsApp message.

```text
Who sent it?      → role
What was sent?    → content
Extra details?    → metadata
```

---

# 🏗️ Structure of a Message

```python
{
    "role": "...",
    "content": "...",
    "metadata": {...}
}
```

Example:

```python
{
    "role": "human",
    "content": "What is LangGraph?",
    "metadata": {
        "timestamp": "10:00 AM"
    }
}
```

---

# 1️⃣ 📋 Role

## What is Role?

Role identifies **who created the message**.

Think of it as:

> "Who is speaking?"

---

## Common Roles

| Role      | Meaning      |
| --------- | ------------ |
| 📋 system | Instructions |
| 👤 human  | User         |
| 🤖 ai     | AI Assistant |
| 🛠️ tool  | Tool Output  |

---

## Example Conversation

```text
📋 System
You are a helpful assistant.

👤 Human
Explain LangChain.

🤖 AI
LangChain is a framework...

🛠️ Tool
Wikipedia search result...
```

Each message has a different role.

---

## Code Example

```python
from langchain_core.messages import HumanMessage

msg = HumanMessage(
    content="What is AI?"
)

print(msg.type)
```

Output:

```text
human
```

---

# 2️⃣ 📝 Content

## What is Content?

Content is the actual message text.

Think of it as:

> "What was said?"

---

## Example

```python
from langchain_core.messages import HumanMessage

msg = HumanMessage(
    content="Explain Agentic AI"
)
```

---

### Access Content

```python
print(msg.content)
```

Output:

```text
Explain Agentic AI
```

---

## Example for AI Message

```python
from langchain_core.messages import AIMessage

msg = AIMessage(
    content="Agentic AI uses reasoning and tools."

```

---

### Access

```python
print(msg.content)
```

Output:

```text

```

# 3️⃣ 🏷️ Metadata

## What is Metadata?

Metadata is additional information about a message.

Think of it as:

> "Extra details about the message."

Not part of the actual conversation.

---

## Real-Life Analogy

WhatsApp message:

```text
Message:
"Hello"

Extra Info:
✓ Time sent
✓ Read receipt
✓ Sender ID
```

Those extra details are metadata.

---

## Example

```python
from langchain_core.messages import HumanMessage

msg = HumanMessage(
    content="What is AI?",
    additional_kwargs={
        "source": "web"
    }
)
```

---

# Complete Example

```python
from langchain_core.messages import HumanMessage
```

# 🤖 AI Message Example

```python
from langchain_core.messages import AIMessage

msg = AIMessage(
    content="LangGraph is used to build agent workflows.",
    response_metadata={
        "model":"gpt-4o",
        "tokens":45
    }
)
```

---

### Access

```python
print(msg.response_metadata)
```

Output:

```python
{
    "model":"gpt-4o",
    "tokens":45
}
```

# 📊 Quick Summary

| Attribute    | Purpose              | Example               |
| ------------ | -------------------- | --------------------- |
| 📋 role      | Who sent the message | human, ai, tool       |
| 📝 content   | Actual message text  | "Explain LangChain"   |
| 🏷️ metadata | Extra information    | tokens, model, source |

---

# 🎯 Easy Way to Remember

```text
💬 Message
   │
   ├── 📋 Role
   │      Who spoke?
   │
   ├── 📝 Content
   │      What was said?
   │
   └── 🏷️ Metadata
          Extra details
```

### One-Line Summary 🚀

> **Every LangChain message contains a role (who sent it), content (the actual message), and metadata (extra information such as model details, token usage, tool calls, timestamps, or custom application data).** 💬📋📝🏷️
