# 🛠️ Tools in LangChain

In LangChain, a **Tool** is anything that allows an AI agent to perform actions outside of pure text generation.

Think of it like this:

```text
🧠 LLM
   +
🛠️ Tools
   =
🦾 AI Agent
```

Without tools:

* AI can only generate text.

With tools:

* 🌐 Search the web
* 📧 Send emails
* 🧮 Calculate
* 📄 Read PDFs
* 💾 Query databases
* 📅 Manage calendars

---

# 1️⃣ Built-in Tools

These are tools provided directly by LangChain or its ecosystem.

---

## 🧮 Python REPL Tool

Allows the AI to execute Python code.

### Install

```bash
pip install langchain-experimental
```

### Example

```python
from langchain_experimental.tools import PythonREPLTool

tool = PythonREPLTool()

result = tool.invoke(
    "print(25 * 40)"
)

print(result)
```

Output:

```text
1000
```

---

## 🐚 Shell Tool

Execute terminal commands.

### Example

```python
from langchain_community.tools import ShellTool

tool = ShellTool()

tool.invoke("pwd")
```

# 🌐 2️⃣ External Tools

These connect to external services and APIs.

---

## 🔍 Tavily Search Tool

Most popular search tool in LangChain.

### Install

```bash
pip install langchain-tavily
```

### Example

```python
from langchain_tavily import TavilySearch

tool = TavilySearch(
    max_results=5
)

response = tool.invoke(
    "Latest AI agent frameworks"
)

print(response)
```

---

### Use with Agent

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(
    model="gpt-4o-mini"
)

agent = model.bind_tools(
    [tool]
)
```

# 🔨 3️⃣ Custom Tools

The most common type in real projects.

You create your own Python functions and expose them as tools.

---

## Using @tool Decorator

### Example

```python
from langchain.tools import tool

@tool
def multiply(a: int, b: int) -> int:
    """
    Multiply two numbers.
    """
    return a * b
```

---

### Invoke

```python
multiply.invoke(
    {"a":25,"b":40}
)
```

Output:

```text
1000
```

---

# Example 2: Weather Tool

```python
from langchain.tools import tool

@tool
def get_weather(city:str):
    """
    Get weather for a city
    """

    return f"Sunny in {city}"
```

---

Usage:

```python
get_weather.invoke(
    {"city":"Chennai"}
)
```

---

# Example 3: Employee Lookup Tool

```python
from langchain.tools import tool

employees = {
    101:"Rahul",
    102:"Priya"
}

@tool
def get_employee(emp_id:int):
    """
    Get employee details
    """

    return employees.get(
        emp_id,
        "Not Found"
    )
```

---

Usage:

```python
get_employee.invoke(
    {"emp_id":101}
)
```

Output:

```text
Rahul
```

# 🔗 Custom Tool + Agent Example

```python
from langchain_openai import ChatOpenAI
from langchain.tools import tool

@tool
def multiply(a:int,b:int):
    """
    Multiply two numbers
    """
    return a*b

model = ChatOpenAI(
    model="gpt-4o-mini"
)

agent = model.bind_tools(
    [multiply]
)

response = agent.invoke(
    "What is 25 multiplied by 40?"
)

print(response)
```

# 🏗️ Multiple Tools Example

```python
from langchain.tools import tool

@tool
def add(a:int,b:int):
    return a+b

@tool
def subtract(a:int,b:int):
    return a-b

@tool
def multiply(a:int,b:int):
    return a*b
```

Bind all tools:

```python
agent = model.bind_tools(
    [
        add,
        subtract,
        multiply
    ]
)
```
