# 📊 `response.usage_metadata` + Cost Calculation

When you invoke a model, LangChain often returns:

```python
response.usage_metadata
```

Example:

```python
response = model.invoke(
    "Explain LangGraph"
)

print(response.usage_metadata)
```

Output:

```python
{
    "prompt_token_count": 100,
    "candidate_token_count": 500,
    "total_token_count": 600
}
```

---

# 📝 Prompt Tokens

Tokens sent **to the model**.

Includes:

```text
📋 System Prompt
👤 User Message
💾 Chat History
🛠️ Tool Results
```

Example:

```text
You are an AI tutor.

Explain LangGraph.
```

Prompt tokens:

```python
{
    "prompt_token_count": 100
}
```

---

# 🤖 Candidate Tokens

Tokens generated **by the model**.

Example output:

```text
LangGraph is a framework used to...
```

```python
{
    "candidate_token_count": 500
}
```

---

# 📊 Total Tokens

```text
Total Tokens
=
Prompt Tokens
+
Candidate Tokens
```

Formula:

```text
total_token_count =
prompt_token_count +
candidate_token_count
```

Example:

```text
100 + 500 = 600
```

---

# 💰 Cost Calculation Formula

Most LLM providers charge separately for:

* 📥 Input tokens
* 📤 Output tokens

General formula:

```text
Total Cost =
(Input Tokens × Input Price)
+
(Output Tokens × Output Price)
```

---

# Formula Using Usage Metadata

```text
Total Cost =
(prompt_token_count × input_token_price)
+
(candidate_token_count × output_token_price)
```

---

# Example 1

Assume model pricing:

```text
Input  = $0.15 per 1M tokens
Output = $0.60 per 1M tokens
```

Usage:

```python
{
    "prompt_token_count": 1000,
    "candidate_token_count": 500
}
```

---

## Input Cost

```text
(1000 / 1,000,000) × 0.15

= $0.00015
```

---

## Output Cost

```text
(500 / 1,000,000) × 0.60

= $0.00030
```

---

## Total Cost

```text
0.00015 + 0.00030

= $0.00045
```

---

# Python Example

```python
usage = response.usage_metadata

input_tokens = usage["prompt_token_count"]
output_tokens = usage["candidate_token_count"]

input_price = 0.15   # per 1M tokens
output_price = 0.60  # per 1M tokens

cost = (
    (input_tokens / 1_000_000) * input_price
    +
    (output_tokens / 1_000_000) * output_price
)

print(cost)
```

# 🧮 Reusable Cost Function

```python
def calculate_cost(
    prompt_tokens,
    output_tokens,
    input_price_per_million,
    output_price_per_million
):
    return (
        (prompt_tokens / 1_000_000)
        * input_price_per_million
    ) + (
        (output_tokens / 1_000_000)
        * output_price_per_million
    )
```

Usage:

```python
cost = calculate_cost(
    prompt_tokens=1000,
    output_tokens=500,
    input_price_per_million=0.15,
    output_price_per_million=0.60
)

print(cost)
```

### Interview Answer 🎤

> `response.usage_metadata` provides token consumption details. `prompt_token_count` represents input tokens, `candidate_token_count` represents output tokens, and `total_token_count` is their sum. LLM cost is calculated using:
>
> **Cost = (Input Tokens × Input Price) + (Output Tokens × Output Price)**
>
> where prices are typically defined per million tokens by the model provider. 🚀📊💰
