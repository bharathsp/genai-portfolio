# 🔗 What is LangChain?

## 1. Define the Concept in Simple Terms

**LangChain is a framework that helps developers build applications powered by Large Language Models (LLMs) like ChatGPT, Gemini, Claude, or Llama.**

A simple definition:

> **🔗 LangChain = A toolkit that connects AI models with tools, data, memory, and workflows.**

Without LangChain, you can ask an AI a question and get an answer.

With LangChain, you can build an AI that can:

* 🌐 Search the web
* 📄 Read PDFs
* 💾 Query databases
* 🧠 Remember previous conversations
* 🛠️ Use external tools
* 🦾 Act as an AI Agent

---

# 🏗️ Why was LangChain created?

An LLM by itself is like a very smart person sitting in an empty room.

🧠 LLM:

> "I can think and answer questions."

But it cannot:

* Search live information
* Read company documents
* Send emails
* Use APIs
* Perform multi-step workflows

LangChain acts like a bridge.

```text
👤 User
    │
    ▼
🔗 LangChain
    │
 ┌──┼─────────────┐
 │  │             │
🧠 LLM      🛠️ Tools
 │              │
💾 Memory   🌐 APIs
 │              │
 📄 Documents
    │
    ▼
✅ Final Answer
```

---

# 🤔 Explain in Simple Terms

Think of LangChain as a **project manager**.

The project manager knows:

* Who should do the work
* Which tools are needed
* What information is required
* How to combine everything

The LLM is the brain.
LangChain coordinates everything.

# 🧩 Main Components of LangChain

---

## 🧠 1. Models

These are the AI brains.

Examples:

* ChatGPT
* Gemini
* Claude
* Llama

Example:

```python
model = ChatOpenAI()
```

---

## 🛠️ 2. Tools

Tools allow AI to interact with the outside world.

Examples:

* 🌐 Web Search
* 📧 Email
* 🧮 Calculator
* 📅 Calendar
* 💾 Database

Example:

```python
model.bind_tools([search, calculator])
```

---

## 💾 3. Memory

Memory helps AI remember previous interactions.

### Without memory

👤:

> My name is Rahul.

👤:

> What is my name?

🤖:

> I don't know.

---

### With memory

👤:

> My name is Rahul.

👤:

> What is my name?

🤖:

> Your name is Rahul.

---

## 📄 4. Document Loaders

These allow AI to read files.

Examples:

* PDF
* Word documents
* CSV
* Excel
* Web pages

Example:

An HR chatbot reads the employee handbook PDF.

---

## ✂️ 5. Text Splitters

Large documents are broken into smaller pieces.

Imagine a 500-page book.

Instead of reading everything at once, LangChain divides it into manageable sections.

---

## 🔍 6. Vector Database (RAG)

Stores document knowledge for fast retrieval.

Example:

Company has:

* Policies
* Product manuals
* Contracts

AI searches only relevant sections.

This is commonly called **RAG (Retrieval-Augmented Generation).**

---

## 🦾 7. Agents

Agents decide:

* Which tool to use
* In what order
* How to solve the problem

Example:

User:

> Book my vacation.

Agent:

* Search flights
* Search hotels
* Calculate budget
* Create itinerary

---

## 🔗 8. Chains

A chain is a sequence of steps.

Example:

```text
User Question
      │
      ▼
Read PDF
      │
      ▼
Search Database
      │
      ▼
Use LLM
      │
      ▼
Generate Answer
```

This sequence is called a **Chain**.

# 💻 Simple LangChain Code Example

Create a model:

```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI()
```

Create a tool:

```python
@tool
def multiply(a: int, b: int):
    return a * b
```

Bind the tool:

```python
agent = model.bind_tools([multiply])
```

Now the AI can use the calculator tool.
