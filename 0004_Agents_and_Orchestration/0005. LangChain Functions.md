# 🔗 Important LangChain Functions & Classes

When building **LLM Apps → Agents → Agentic Workflows**, there are a few LangChain functions and classes you'll use repeatedly.

---

# 🏗️ 1. `init_chat_model()`

## What is it?

A generic way to initialize a chat model.

Instead of importing provider-specific classes, you can initialize different models using the same function.

### Example

```python
from langchain.chat_models import init_chat_model

model = init_chat_model(
    "gpt-4o-mini",
    model_provider="openai"
)
```

---

### Why use it?

Without `init_chat_model()`:

```python
ChatOpenAI(...)
ChatGoogleGenerativeAI(...)
ChatGroq(...)
```

Different code for different providers.

With `init_chat_model()`:

```python
model = init_chat_model(...)
```

Same code pattern regardless of provider.

---

### Real-life analogy 🚗

Think of it as renting a car.

```text
🚙 Toyota
🚗 Honda
🏎️ Ferrari
```

Instead of learning different controls for each vehicle, you get a standard dashboard.

# 🛠️ 5. `@tool`

## What is it?

One of the most important concepts in Agentic AI.

The `@tool` decorator converts a Python function into a tool an AI agent can use.

---

### Example

```python
from langchain.tools import tool

@tool
def multiply(a: int, b: int) -> int:
    """Multiply two numbers"""
    return a * b
```

---

Now the LLM can use it.

```python
agent = model.bind_tools(
    [multiply]
)
```

---

### User asks

```text
What is 25 × 40?
```

The agent can call:

```python
multiply(25, 40)
```

and return:

```text
1000
```

---

### Real-life analogy 🔨

Without tools:

👨‍🔧 Worker knows theory.

With tools:

👨‍🔧 + 🔨 + 🔧 + 🪛

Can actually perform work.

# ⛓️ 12. LangChain Expression Language (LCEL)

Create chains using `|`

### Example

```python
chain = prompt | model
```

---

More advanced:

```python
chain = (
    prompt
    | model
    | output_parser
)
```

---

### Real-life analogy

Assembly line.

```text
Input
  ↓
Prompt
  ↓
LLM
  ↓
Parser
  ↓
Output
```

# 🚀 Mini Agent Example

```python
from langchain_openai import ChatOpenAI
from langchain.tools import tool

@tool
def multiply(a:int,b:int):
    return a*b

model = ChatOpenAI(
    model="gpt-4o-mini"
)

agent = model.bind_tools(
    [multiply]
)

response = agent.invoke(
    "What is 25 times 40?"
)
```
