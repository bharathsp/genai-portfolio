# 📚 Skill.md in MCP (Model Context Protocol)

As MCP ecosystems evolve, you'll often hear about:

```text
skill.md
```

This is especially common in MCP servers, AI coding assistants, and agent platforms.

---

# 🎯 What is skill.md?

## Simple Definition

> **skill.md is a Markdown file that teaches the LLM how to use a specific skill, tool, workflow, or capability.**

Think of it as:

```text
Tool
  +
Documentation
  +
Instructions
  =
skill.md
```

---

# 🏠 Real-Life Analogy

Imagine a new employee joins a company.

You don't just give them access to:

```text
🖥️ Laptop
📧 Email
📊 Dashboard
```

You also provide:

```text
📖 Training Guide
```

which explains:

```text
What it does

When to use it

How to use it

Best practices
```

A **skill.md** is that training guide for an AI.

---

# Why Do We Need skill.md?

Suppose an MCP Server exposes:

```python
create_invoice()

send_email()

generate_report()
```

Without documentation the LLM may not know:

❌ When to use them

❌ Required inputs

❌ Workflow order

❌ Business rules

---

With skill.md:

```text
To generate an invoice:

1. Validate customer
2. Create invoice
3. Email invoice

Use create_invoice() first.
```

The LLM becomes much smarter.

```markdown
# Travel Assistant Skill

Use this skill for travel planning.

Available Tools:

- search_flights
- search_hotels
- create_itinerary

Workflow:

1. Search flights
2. Search hotels
3. Create itinerary

Do not create itineraries
without flight information.
```

# skill.md vs Tool Description

Many beginners confuse these.

---

## Tool Description

```python
@tool
def search_flights():
    """Search available flights"""
```

Describes:

```text
Single Tool
```

---

## skill.md

Explains:

```text
Entire Workflow

Tool Relationships

Business Rules

Examples

Best Practices
```

---

# Comparison

| Feature            | Tool Description | skill.md |
| ------------------ | ---------------- | -------- |
| Explains Tool      | ✅                | ✅        |
| Explains Workflow  | ❌                | ✅        |
| Multi-Step Process | ❌                | ✅        |
| Business Rules     | ❌                | ✅        |
| Examples           | Limited          | ✅        |
| Best Practices     | ❌                | ✅        |

# Interview Answer 🎤

> `skill.md` is a Markdown-based instruction file used in MCP ecosystems to describe a skill, workflow, or capability exposed by an MCP server. It provides guidance to the LLM about when to use tools, how tools relate to each other, workflow sequences, business rules, examples, and best practices. While tool descriptions explain individual tools, `skill.md` explains the overall process and helps the agent perform more accurate and consistent multi-step reasoning.
