# 🔍 What is LangSmith?

## 1. Define the Concept in Simple Terms

**LangSmith is a platform that helps developers debug, monitor, test, and improve AI applications and AI agents.**

A simple definition:

> **🔍 LangSmith = A monitoring and debugging platform for AI applications.**

If LangChain helps you **build** AI applications,

then LangSmith helps you **observe, analyze, test, and improve** them.

Think of it as:

```text
🔗 LangChain  → Build AI Apps

🌳 LangGraph → Build Complex AI Workflows

🔍 LangSmith → Monitor and Debug AI Workflows
```

---

# 🤔 Why was LangSmith created?

Imagine you build an AI travel assistant.

User:

> Plan my Europe trip.

The AI:

* Searches flights
* Searches hotels
* Calculates budget
* Creates itinerary

But something goes wrong.

Maybe:

* ❌ Flight API failed
* ❌ Wrong hotel selected
* ❌ Budget calculation incorrect
* ❌ AI used the wrong tool

How do you find the problem?

👉 LangSmith records the entire process.

# 🏗️ Main Features of LangSmith

---

## 📜 1. Tracing

Tracing means recording every step the AI performs.

Example:

```text
👤 User:
Plan my Goa trip.

        │
        ▼
🧠 LLM understands request

        │
        ▼
🌐 Flight API called

        │
        ▼
🏨 Hotel API called

        │
        ▼
💰 Budget calculation

        │
        ▼
✅ Final response
```

LangSmith stores this complete trace.

---

## 🐞 2. Debugging

Suppose the AI gives a wrong answer.

LangSmith helps answer:

* Which tool failed?
* Which API returned incorrect data?
* Which prompt caused the issue?

Example:

```text
❌ Hotel API Timeout

Result:
No hotels found.

LangSmith:
Shows exactly where the error happened.
```

---

## 🧪 3. Testing

You can test your AI application before releasing it.

Example test cases:

| User Question      | Expected Result       |
| ------------------ | --------------------- |
| What is 2+2?       | 4                     |
| Summarize this PDF | Correct summary       |
| Book a hotel       | Hotel booking created |

LangSmith can run these tests automatically.

---

## 📊 4. Monitoring

After deployment, LangSmith tracks:

* ⚡ Response time
* 💲 Token usage
* 💰 API cost
* ❌ Error rate
* 📈 Performance trends

Example:

You discover:

* Yesterday: 2% failures
* Today: 15% failures

This helps identify problems quickly.

---

## 📈 5. Evaluation

LangSmith helps measure AI quality.

Questions like:

* Is the answer accurate?
* Is the summary complete?
* Is the response relevant?

Developers can compare different AI models.

Example:

Model A:
Accuracy = 90%

Model B:
Accuracy = 96%

Choose the better one.

# 💻 Simple LangSmith Code Example

Without LangSmith:

```python
result = chain.invoke(
    "Summarize this PDF"
)
```

With LangSmith enabled:

```python
import os

os.environ["LANGCHAIN_TRACING_V2"] = "true"
os.environ["LANGCHAIN_API_KEY"] = "your-api-key"

result = chain.invoke(
    "Summarize this PDF"
)
```

Now LangSmith automatically records the workflow.
